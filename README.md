# Kalkulator średniej z edukacja.cl


Pobierz wszystkie wymagane biblioteki (plik `requirements.txt`) za pomocą polecenia:
```
pip3 install -r requirements.txt
```

Stwórz plik `.env`

Uzupełnij go swoimi danymi:
```
USERNAME=pwr123456
PASSWORD=hasło
```

Uruchom skrypt:
```
python3 EdukacjaCl.py
```
